# utopia_cms_ui_example

A design-system sheet for `utopia_cms_ui`: every token (colors, typography,
shape) and every component (buttons, fields, selection controls, chips,
loading states, dialogs, table, sidebar) laid out on one themable page, so a
change to a widget or a theme value is visible across the whole catalog at
once. A theme picker at the top switches between the same 5 modes as the
`utopia_cms` example.

## Running

From the workspace root:

```
melos run run:ui-example
```

Or directly:

```
cd packages/ui/example && flutter run -d chrome
```
